# 🎓 Student Performance Prediction using Machine Learning

## 📌 Introduction
This project is a Mini AI/ML project that predicts whether a student will pass or fail based on their academic performance using Machine Learning algorithms.

---

## 🎯 Objective
The main objective of this project is to build a model that can predict student results using:
- Study Hours
- Attendance
- Previous Marks

---

## 📊 Dataset
The dataset used in this project contains:
- Study Hours
- Attendance
- Previous Marks
- Output: Pass (1) / Fail (0)

---

## ⚙️ Tools and Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🔍 Methodology
1. Data Collection  
2. Data Cleaning  
3. Data Visualization  
4. Splitting Data (Train/Test)  
5. Model Training  
6. Model Evaluation  

---

## 🤖 Models Used
- Logistic Regression  
- Decision Tree Classifier  

---

## 📈 Results
- Logistic Regression Accuracy: ~85%  
- Decision Tree Accuracy: ~90%  

---

## 📊 Evaluation Metrics
- Accuracy Score  
- Confusion Matrix  
- Classification Report  

---

## 💡 Conclusion
This project shows that Machine Learning can be used to predict student performance effectively. Decision Tree model gave better accuracy compared to Logistic Regression.
